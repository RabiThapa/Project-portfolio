How does Internet Works?

What is Internet?

The Internet is the backbone of the Web, the technical infrastructure that makes the Web possible. At its most basic,the Internet is a large network of computers which communicate all together. Internet is a way to connect computers all together and ensure that, whatever happens, they find a way to stay connected.

How does internet works?

When two computers need to communicate, you have to link them, either physically (usually with an Ethernet cable) or wirelessly (for example with Wi-Fi or Bluetooth systems).All modern computers can sustain any of those connections.Such a network is not limited to two computers. You can connect as many computers as you wish.

Each computer on a network is connected to a special tiny computer called a router. Router makes sure that a message sent from a given computer arrives at the right destination computer.

A network of networks

What about connecting hundreds, thousands, billions of computers? Of course a single router can't scale that far, but, router is a computer like any other, we can connect as many computers as you wish. Connecting computers to routers, then routers to routers, we are able to scale infinitely.

Finding Computers

If you want to send a message to a computer, you have to specify which one. Thus any computer linked to a network has a unique address that identifies it, called an "IP address". IP address is made of a series of four numbers separated by dots, for example: 192.168.2.10
But we human beings have a hard time remembering that sort of address. To make things easier, we can alias an IP address with a human readable name called a domain name. For example (at the time of writing; IP addresses can change) google.com. Is is the domain name used on top of the IP address 142.250.190.78.

Internet and the Web

The Internet is an infrastructure, whereas the Web is a service built on top of the infrastructure. It is worth noting there are several other services built on top of the Internet, such as email and IRC.
