What is version control?

The management of changes to documents, computer programs, large websites, and other collections of information.

What is Git?

Git is a version control system which lets you track changes you make to your files over time. With Git, you can revert to various states of your files (like a time traveling machine). You can also make a copy of your file, make changes to that copy, and then merge these changes to the original copy.

What is GitHub?

Git Hub is the website where you host all of your Git repositories. Being online, it makes easy to work in groups with the others people, and organize your projects into a portfolio. Git is the tool that tracks the changes in your code over time.

Repository

Project, or the folder/ place where your project is kept.

Commands in Git

  1. git init (Inside of the project folder)
  2. git add<"filename"> or git add( To add file)
  3. git commit -m "Comments"(Create commit point)
  4. git config --list
  5. git config --list
  6. git config --global user.name" "
  7. git config --global user.email" "
  8. git checkout -b<"branch name"> (Create new branch)
  9. git checkout<"branch_name"> (Move across branch)
  10. git branch(branch_list)
  11. git merge master (Merge changes in the master to branch)
  12. git merge<"branch name">(Merge the changges in the branch to master)

Into Github

  1. git remote add origin< "link" >
  2. git push -u origin master (To push)
  3. git pull -u origin master (To pull)
